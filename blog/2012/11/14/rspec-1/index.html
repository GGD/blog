Rspec(1)

Rspec 是用來測試的一個套件，這邊簡單筆記一下教學

SETUP

$ gem install rspec

接著在專案目錄底下執行
$ rspec —init
create spec/spec_helper.rb
create .rspec

DESCRIBE

假設目前的專案底下有個 lib/zombie.rb
因此相對應的 spec 會位於 spec/lib/zombie_spec.rb
在此 spec 底下，首先要先 require spec_helper，當中包含了 spec 的各項設定。


接著要敘述 zombie 的行為，因此要寫個 describe block，在 block 裡面則撰寫 zombie 的各種 example（這裡不稱作 test）。

此例當中，should 稱為 modifier，< 稱為 matcher。
modifier 和 matcher 可以有多種寫法，也可以自己寫一套。下述幾種都算常見：
zombie.name.should == ‘Ash
zombie.alive.should == false # 可寫成：zombie.alive.should be_false（be 是 matcher）
zombie.rotting.should == true # 可寫成：zombie.rotting.should be_true
zombie.height.should >= 5
zombie.brain.should be < 1
zombie.height.should_not == 5

而針對判斷式，spec 的寫法可以更 human：
zombie.should be_hungry

這段 spec 執行後也可以 pass。
