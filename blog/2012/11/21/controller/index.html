to_json

response 特定欄位：
@zombie.to_json(only: :name)
@zombie.to_json(only: [:name, :age])
response 某些欄位拿掉：
@zombie.to_json(except: [:created_at, updated_at, :id])
responce 包含 relation：
@zombie.to_json(include: :brain, except: [:created_at, updated_at] )
