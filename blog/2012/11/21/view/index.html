link_to

<%= link_to post.user, user_path(post.user) %>

可以縮寫成
<%= link_to post.user, post.user %>

nested routes

<%= link_to 'Show', user_path(@user, post) %>

可以縮寫成
<%= link_to ‘Show’, [@user, post] %>

特殊對應欄位

如果 browser 有支援（比如 iphone），那游標在底下這些欄位時會有對應的或控制項：
<%= f.password_field :password %>
<%= f.number_field :price %>
<%= f.range_field :quantity %>
<%= f.email_field :email %>
<%= f.url_field :website %>
<%= f.telephone_field :mobile %>

div_for

<% @tweets.each do |tweet| %>
  <div id="tweet_<%= tweet.id %>" class="tweet">
    <%= tweet.body %>
  </div>
<% end %>

等同於
<% @tweets.each do |tweet| %>
<%= div_for tweet do %>
<%= tweet.body %>
<% end %>
<% end %>
