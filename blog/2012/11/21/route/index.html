Route

Route 筆記。

redirect to

http://localhost:3000/all redirect to http://localhost:3000/books
match ‘all’ => redirect(‘/books’)

external link

match 'google' => redirect('http://www.google.com/')

Routes with Parameters


/local_tweet/32828 could be done by
match ‘local_tweet/:zipcode’ => ‘tweets#index’

further

match 'local_tweet:zipcode' => 'tweets#index', :as => 'local_tweets'

對應的 link_to 寫法：
<%= link_to “Tweets in 32828”, local_tweets_path(32828) %>
