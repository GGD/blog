Symbol#to_proc

to_proc() public
Turns the symbol into a simple proc, which is especially useful for enumerations. Examples:

Another example

tags.map(&:name).join(' '), the &:name means…

The & is a shortcut to Symbol#to_proc which will convert the symbol you pass to it to a method name on the object. So &:name converts to { |reciever| receiever.name } which is then passed to the map method.
So above is a shortcut for tags.map(:name.to_proc) which is like calling tags.map{|tag| tag.name } and just collects all the tag names into an array.
