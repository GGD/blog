Scss - variable

Scss - variable。

variable type

Boolean

$rounded: false;
$shadow: true;

Numbers (可寫單位)

$rounded: 4px;
$line-height: 1.5;
$font-size: 3rem;

Colors

$base: purple;
$border: rgba(0, 255, 0, 0.5);
$shadow: #333;

Strings

$header: 'My Head';
$callout: Arial;
$message: "Oops...";

Lists

$authors: Mike, John, Tom;
$margin: 40px 0 20px 100px;

Null

$shadow: null;
